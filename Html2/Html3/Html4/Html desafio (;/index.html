⭒ Como criei meu primeiro site

Voltar

Criei meu primeiro site HTML após aprender os fundamentos no curso da DIO. Utilizei o VSCode como editor de texto. Criei uma pasta e fiz o arquivo index.html, escrevi a estrutura básica do HTML, incluindo cabeçalho e corpo.
No cabeçalho, defini o título da página. No corpo, inseri título, linha de separação, parágrafos, listas e links explicando resumidamente o que aprendi. Depois de salvar o arquivo, visualizei o site no navegador e fiquei super animada com o resultado!
Esse projeto me deu a base para continuar explorando o desenvolvimento web e crescer mais nisso.

⭒ Utilizando as tags

Voltar

Para criar uma página HTML, usei várias tags que aprendi no curso. A tag "title" define o título da página que aparece na aba do navegador e deve ser colocada dentro da tag "head". As tags "h1", "h2", e "h3" são usadas para adicionar cabeçalhos de diferentes tamanhos.
O "h1" sendo o maior, geralmente usado para título principal, enquanto "h2" e "h3" são usados para subtítulos. A tag "p" define parágrafos, onde você escreve textos simples para explicar o título.
Para criar links para outras páginas ou recursos, usei a tag "a", que precisa do atributo "href" para especificar o destino do link.
Para listas, existem as tags "ul" e "ol". A tag "ul" cria uma lista não ordenada, com marcadores, enquanto "ol" cria uma lista ordenada, numerada.
Cada item da lista é definido com a tag "li". Essas tags são fundamentais para criar uma página fácil de navegar.

⭒ O que é o HTML

Voltar

O HTML é usado para criar sites/páginas, organizando conteúdos como textos, imagens, links para outras páginas, listas, etc usando as tags. Dentro existe o cabeçalho "head" e o corpo "body".
Não é uma linguagem de programação, mesmo o nome sendo HyperText Markup Language, o HTML é uma linguagem de marcação que serve para descrever e estruturar o conteúdo da web.
Mas ainda sim, é ótima para quem está começando nessa área da programação, já que é simples de usar e não tem códigos complexos para gerar conteúdos.
Então, basicamente, o HTML é a base para construir qualquer página da web, organizando e apresentando o conteúdo que as pessoas veem e interagem.
Sem o HTML, não seria possível criar páginas web bem estruturadas e funcionais. Ele é essencial para garantir que o conteúdo seja exibido de maneira certa e também acessível. Além disso, o HTML é suportado por todos os navegadores!
